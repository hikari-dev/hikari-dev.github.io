最近的一些catch-ups

2020-12-08
生活
次阅读

文章目录

之前用 ktor 写了个 setu 后端，本来想再搞个安卓前端啥的用来展示一下，后来忙别的事情去了也就不了了之了，暂且放到TODO List，等以后有时间（兴趣）了再说吧咕咕咕~

上个月被选为替补参加了公司举办的篮球赛，一路跟着竟然混进了这个月初的决赛，运气不错还拿了个冠军，虽然没啥实质性的奖励，但是也算是一个小小的荣誉了吧。

这几个周末的大多数时间都用来玩刺客（无双）信条了，终于把所有支线都做完了，游戏是个好游戏，就是对于我这种强迫症来说，有点肝。

刚刚把博客从 Hexo 迁移到了 Hugo ，要说原因呢，主要还是 Hugo 是用 go 写的，构建速度比 Hexo 快了不少，而且这次加了CI配合 Hugo 使用，以后发布新文章方便了不少。迁移之路总体上还是比较平坦的，就是加评论功能的时候给评论模块添加 loading 样式遇到了点问题，果然全靠搜索引擎学习的 css 是真的8太行啊，还好最后在借鉴（抄）一位大佬的博客后解决了这个问题。之前用的 Butterfly 主题感觉有点花哨，这次换了这个 Even 主题，感觉简洁了不少。

11月第一次 leetcode 全勤（用了两张补签卡），拿到了个勋章，感觉挺开心的，以后争取能多拿几个吧。

不知道无限列车啥时候能看，最近比较想看的剧就这一部了，当然，还有巨人最终季。

听说茶颜悦色挺好喝的，有机会还蛮想品一下。
