Compose for Desktop 初体验

2021-03-09
瞎搞
次阅读

文章目录

其实在 Compose for Desktop 刚出道的时候就简单上手体验了下，声明式UI写起来着实感觉挺爽的，但是当时在功能上还有许多不完善的地方，就没深入学习了。现在 Compose for Desktop 终于迎来了它的第三个里程碑版本，添加了很多对桌面平台的兼容支持，至少 TextFiled 终于可以使用键盘快捷键了（什么，以前竟然不能 CtrlCV 快速复读？！

试了一下用 Compose for Desktop 做了个批量重命名工具，简单起见，都是用的默认样式（其实是还没学自定义样式)。

看起来也还行吧，反正功能是达到了，至少默认的 MaterialDesign 样式比 swing 那套默认样式还是要好看一点的。

代码放在 我的github 上了，有需要的可以去看一下。
